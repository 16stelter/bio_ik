# BioIK Inverse Kinematics Plugin for MoveIt




## Parameters
### End Effector Parameters
- `position_only_ik` (just like the `position_only_ik` parameter already provided by some other MoveIt IK solvers)
- `weight`

### IK Group Parameters
- `population_size`
- `elite_count`
- `quadratic_error`
- `full_circle`
- `no_wipeout`
- `fitness_randomization`
- `linear_adjustment`
